# 🔀 归并排序

> 📝 经典排序算法 | 时间复杂度 O(n log n) | 空间复杂度 O(n)

---

## 📖 算法思想

采用分治法，将数组分成两半，分别排序，然后将两个有序数组合并成一个有序数组。

---

## 💡 特点

- ✅ 时间复杂度稳定为 O(n log n)
- ✅ 稳定排序
- ❌ 需要额外的 O(n) 空间
- 💡 适合链表排序或外部排序

---

## 🔧 代码实现

```java
/**
 * @description: 归并排序
 * @author: minalz
 * @date: 2021-06-08 00:52
 **/
public class MergeSort {

    /**
     * 归并排序 涉及到递归
     * @param array
     */
    public void mergeSort(int[] array) {
        int[] temp = new int[array.length];
        mergeSortImpl(array, 0, array.length - 1, temp);
    }

    public void mergeSortImpl(int[] array, int start, int end, int[] temp) {
        if (start >= end) {
            return;
        }
        int mid = (start + end) / 2;
        mergeSortImpl(array, start, mid, temp);
        mergeSortImpl(array, mid + 1, end, temp);
        merge(array, start, mid, end, temp);
    }

    public void merge(int[] array, int start, int mid, int end, int[] temp) {
        int left = start;
        int right = mid + 1;
        int index = start;
        while (left <= mid && right <= end) {
            if (array[left] < array[right]) {
                temp[index++] = array[left++];
            } else {
                temp[index++] = array[right++];
            }
        }
        while (left <= mid) {
            temp[index++] = array[left++];
        }
        while (right <= end) {
            temp[index++] = array[right++];
        }
        for (index = start; index <= end; index++) {
            array[index] = temp[index];
        }
    }

    @Test
    public void testMerge() {
        int[] array = new int[10];
        for (int i = 0; i < 10; i++) {
            array[i] = (int) (Math.random() * 10000);
        }

        mergeSort(array);

        for (int i = 0; i < array.length; i++) {
            System.out.print(array[i] + ",");
        }
    }
}
```
