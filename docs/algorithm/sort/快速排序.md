# ⚡ 快速排序

> 📝 经典排序算法 | 平均时间复杂度 O(n log n) | 空间复杂度 O(log n)

---

## 📖 算法思想

选择一个基准元素（pivot），将数组分为两部分：左边都小于 pivot，右边都大于 pivot。然后递归地对两部分进行排序。

---

## 💡 特点

- ✅ 平均时间复杂度优秀
- ✅ 实际应用中效率很高
- ❌ 最坏情况 O(n²)
- ❌ 不稳定排序

---

## 🔧 代码实现

```java
/**
 * @description: 快速排序（时间复杂度 O(n log n)）
 * @author: minalz
 * @date: 2021-06-08 00:39
 **/
public class QuickSort {

    /**
     * 快速排序
     * @param array
     */
    public void quickSort(int[] array) {
        sort(array, 0, array.length - 1);
    }

    /**
     * 递归方法
     * @param array
     * @param start
     * @param end
     */
    public void sort(int[] array,int start, int end) {
        if (start >= end) {
            return;
        }
        int pivot = array[start];
        int left = start;
        int right = end;
        while (left <= right) {
            while (left <= right && array[left] < pivot) {
                // 找到左边第一个大于pivot的值
                left++;
            }
            while (left <= right && array[right] > pivot) {
                // 找到右边第一个小于pivot的值
                right--;
            }
            if (left <= right) {
                int temp = array[left];
                array[left] = array[right];
                array[right] = temp;
                left++;
                right--;
            }
        }
        // 左半
        sort(array, start, right);
        // 右半
        sort(array, left, end);
    }

    @Test
    public void testQuick() {
        int[] array = new int[10];
        for (int i = 0; i < 10; i++) {
            array[i] = (int) (Math.random() * 10000);
        }

        quickSort(array);

        for (int i = 0; i < array.length; i++) {
            System.out.print(array[i] + ",");
        }
    }
}

```
