# ✂️ 单词拆分

> 📝 LeetCode 139. 单词拆分 | 动态规划 | 字符串处理

---

## 📖 题目描述

给定一个非空字符串 `s` 和一个包含非空单词列表的字典 `wordDict`，判定 `s` 是否可以被空格拆分为一个或多个在字典中出现的单词。

---

## 💡 解题思路

- 使用动态规划，`canSegment[i]` 表示前 `i` 个字符是否可以被拆分
- 状态转移：`canSegment[i] = true`，如果存在 `j` 使得 `canSegment[i-j] = true` 且 `s[i-j..i-1]` 在字典中
- 优化：只需遍历到字典中最长单词的长度

---

## 🔧 代码实现

```java
/**
 * @description: 139. 单词拆分
 * @author: minalz
 * @date: 2021-06-05 23:16
 **/
public class WordBreak {

    public boolean wordBreak(String s, List<String> wordDict) {
        Set<String> dict = new HashSet<>(wordDict);
        boolean[] canSegment = new boolean[s.length() + 1];
        canSegment[0] = true;
        int largestLengthWord = getLargest(dict);
        for (int i = 1; i <= s.length(); i++) {
            for (int j = 0; j <= largestLengthWord && j <= i; j++) {
                if (!canSegment[i - j]) {
                    continue;
                }
                if (dict.contains(s.substring(i - j, i))) {
                    canSegment[i] = true;
                }
            }
        }
        return canSegment[s.length()];
    }

    public int getLargest(Set<String> dict) {
        int max = 0;
        for (String word: dict) {
            max = Math.max(max, word.length());
        }
        return max;
    }
}

```
