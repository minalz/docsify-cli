# ⬅️ 二叉树的后序遍历

> 📝 LeetCode 145. 二叉树的后序遍历 | DFS | 递归 / 迭代

---

## 📖 题目描述

给定一个二叉树的根节点 `root`，返回它的后序遍历（左-右-根）。

---

## 💡 解题思路

### 方法一：递归

- 递归遍历左子树
- 递归遍历右子树
- 访问根节点

### 方法二：非递归（改变树结构）

- 使用栈，利用改变树结构的方式标记已访问节点

### 方法三：非递归（使用 prev 指针）

- 使用栈和 `prev` 指针记录上次访问的节点
- 判断当前节点的子节点是否已访问

---

## 🔧 代码实现

```java
/**
 * @description: 二叉树的后序遍历
 * @author: minalz
 * @date: 2021-05-25 23:04
 **/
public class PostOrderTree {

    /**
     * 递归方式
     * @param root
     * @return
     */
    public List<Integer> postorderTraversal(TreeNode3 root) {
        List<Integer> result = new ArrayList<>();
        if(root == null) {
            return result;
        }

        List<Integer> left = postorderTraversal(root.left);
        List<Integer> right = postorderTraversal(root.right);

        result.addAll(left);
        result.addAll(right);
        result.add(root.val);

        return result;
    }

    /**
     * 非递归方式 -- 会改变原来树的数据结构
     * @param root
     * @return
     */
    public List<Integer> postorderTraversal1(TreeNode3 root) {
        List<Integer> result = new ArrayList<>();
        if(root == null) {
            return result;
        }

        Stack<TreeNode3> stack = new Stack<>();
        stack.push(root);
        while (!stack.isEmpty()) {
            TreeNode3 node = stack.peek();

            if (node.right == null && node.left == null) {
                result.add(stack.pop().val);
            }
            if (node.right != null) {
                stack.push(node.right);
                node.right = null;
            }
            if (node.left != null) {
                stack.push(node.left);
                node.left = null;
            }
        }

        return result;
    }

    /**
     * 非递归方式
     * @param root
     * @return
     */
    public List<Integer> postorderTraversal2(TreeNode3 root) {
        List<Integer> result = new ArrayList<>();
        if(root == null) {
            return result;
        }

        Stack<TreeNode3> stack = new Stack<>();
        TreeNode3 prev = null;
        TreeNode3 curr = root;
        stack.push(root);

        while (!stack.isEmpty()) {
            curr = stack.peek();
            if (prev == null || prev.left == curr || prev.right == curr) {
                if (curr.left != null) {
                    stack.push(curr.left);
                } else if (curr.right != null) {
                    stack.push(curr.right);
                }
            } else if (curr.left == prev) {
                if (curr.right != null) {
                    stack.push(curr.right);
                }
            } else {
                result.add(curr.val);
                stack.pop();
            }
            prev = curr;
        }

        return result;
    }
}

// Definition for a binary tree node.
class TreeNode3 {
    int val;
    TreeNode3 left;
    TreeNode3 right;

    TreeNode3() {
    }

    TreeNode3(int val) {
        this.val = val;
    }

    TreeNode3(int val, TreeNode3 left, TreeNode3 right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}
```
