# 🔍 二分查找法

> 📝 经典二分查找算法 | 时间复杂度 O(log n)

---

## 📖 算法思想

在有序数组中，通过每次比较中间元素，将搜索范围缩小一半，直到找到目标值或搜索范围为空。

---

## 💡 关键点

- ✅ 数组必须是有序的
- ✅ 注意循环终止条件（`start + 1 < end`）
- ✅ 避免整数溢出（`mid = start + (end - start) / 2`）
- ✅ 最后单独判断 `start` 和 `end` 位置

---

## 🔧 代码实现

```java
public class BinarySearch {

    @Test
    public void test01(){
        int[] nums = {1,3,4,6,8,33,55,57,66,78,94,100};
        int index = getIndex(nums,66);
        System.out.println(index);
    }

    public int getIndex(int[] nums, int value){
        if(nums == null || nums.length == 0){
            return -1;
        }
        int start = 0;
        int end = nums.length - 1;
        int mid = 0;
        while (start + 1 < end){
            mid = start + (end - start)/2;
            if(nums[mid] == value){
                return mid;
            }else if(nums[mid] > value){
                end = mid;
            }else{
                start = mid;
            }
        }
        if(nums[start] == value){
            return start;
        }
        if(nums[end] == value){
            return end;
        }
        return mid;
    }
}
```
