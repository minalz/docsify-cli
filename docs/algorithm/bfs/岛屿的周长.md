# 🌊 岛屿的周长

> 📝 LeetCode 463. 岛屿的周长 | DFS | 网格遍历

---

## 📖 题目描述

给定一个二维网格地图，其中 `1` 表示陆地，`0` 表示水域。网格被水完全包围，且恰好有一个岛屿。计算这个岛屿的周长。

---

## 💡 解题思路

- 遍历网格找到第一个陆地格子
- 使用 DFS 计算周长：
  - 超出网格范围 → 周长 +1（黄色边）
  - 遇到水域 → 周长 +1（蓝色边）
  - 已遍历的陆地 → 周长 +0

---

## 🔧 代码实现

```java
/**
 * @description: 463. 岛屿的周长
 * @author: minalz
 * @date: 2021-05-30 22:55
 **/
public class BFSDemo2 {

    public int islandPerimeter(int[][] grid) {
        if (grid == null || grid.length == 0) {
            return 0;
        }
        int row = grid.length;
        int col = grid[0].length;
        for (int i = 0; i < row; i++) {
            for (int j = 0; j < col; j++) {
                if (grid[i][j] == 1) {
                    return bfs(grid, i, j);
                }
            }
        }
        return 0;
    }

    public int bfs(int[][] grid, int i, int j) {

        // 函数因为「坐标 (r, c) 超出网格范围」返回，对应一条黄色的边
        if (!inArea(grid, i, j)) {
            return 1;
        }
        // 函数因为「当前格子是海洋格子」返回，对应一条蓝色的边
        if (grid[i][j] == 0) {
            return 1;
        }
        // 函数因为「当前格子是已遍历的陆地格子」返回，和周长没关系
        if (grid[i][j] != 1) {
            return 0;
        }

        grid[i][j] = 2;

        return bfs(grid, i - 1, j) +
                bfs(grid, i + 1, j) +
                bfs(grid, i, j - 1) +
                bfs(grid, i, j + 1);
    }

    public boolean inArea(int[][] grid, int i, int j) {
        return i >= 0 && j >= 0 && i < grid.length && j < grid[0].length;
    }

    @Test
    public void test01() {
        int[][] grid = {{0,1,0,0},{1,1,1,0},{0,1,0,0},{1,1,0,0}};
        System.out.println(islandPerimeter(grid));
    }
}

```
