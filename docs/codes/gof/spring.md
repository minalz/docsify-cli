# Spring中的设计模式

| 设计模式名称 | 涉及到的java类        |
| :----------- | --------------------- |
| 工厂模式     | BeanFactory           |
| 装饰器模式   | BeanWrapper           |
| 代理模式     | AopProxy              |
| 委派模式     | DispatcherServlet     |
| 策略模式     | HandlerMapping        |
| 适配器模式   | HandlerAdapter        |
| 模版方法     | JdbcTemplate          |
| 观察者模式   | ContextLoaderListener |
