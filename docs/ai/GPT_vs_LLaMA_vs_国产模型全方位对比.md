### 3.2 关键技术组件

| 技术点 | GPT-3 | GPT-5 | LLaMA-3 | Qwen3 | DeepSeek-V3 | Kimi K2 |
|--------|-------|-------|---------|-------|------------|---------|
| 位置编码 | 绝对位置编码 | 未知 | RoPE | RoPE + YaRN | RoPE | RoPE |
| 注意力机制 | MHA | 未知 | GQA | GQA | **MLA（多头潜在注意力）** | GQA |
| 归一化 | LayerNorm | 未知 | RMSNorm | RMSNorm | RMSNorm | RMSNorm |
| 激活函数 | GeLU | 未知 | SwiGLU | SwiGLU | SwiGLU | SwiGLU |
| MoE 结构 | ❌ | 未知 | ❌ | ✅ 128专家 | ✅ DeepSeekMoE | ✅ |
| KV Cache 优化 | 标准 | 未知 | 标准 | 标准 | **MLA 压缩 75%** | 标准 |
| 多 token 预测 | ❌ | 未知 | ❌ | ❌ | ✅（训练加速） | ❌ |
| 推理模式切换 | ❌ | ✅ 自动路由 | ❌ | ✅ 思考/非思考 | R1 版本支持 | ✅ Thinking |

> **DeepSeek MLA（Multi-head Latent Attention）** 是 DeepSeek-V2/V3 的核心创新：
> 将 Key/Value 投影到低维潜在空间，KV Cache 压缩约 75%，推理显存需求大幅降低，是其低成本部署的关键。

| GPT-4o | RLHF + RLAIF + 红队 | 强安全对齐 |

## 💡 十一、各模型核心创新与差异化

### Qwen3（阿里巴巴）
```
核心亮点：
✅ 思考/非思考双模式 — 同一模型支持快速回答和深度推理，按需切换
✅ 128 专家 MoE — 激活比例极低（22B/235B ≈ 9.4%），推理成本极低
✅ 119 语言支持 — 国际化程度最高的中国模型
✅ 36T tokens 训练 — 国内训练数据量最大
✅ Qwen3-VL — 多模态视觉理解，原生 256K 上下文
```

### DeepSeek（深度求索）
```
核心亮点：
✅ MLA（Multi-head Latent Attention）— KV Cache 压缩 75%，推理显存大幅降低
✅ DeepSeekMoE — 更细粒度专家划分，每层 256 专家，每次激活 8 个
✅ 多 Token 预测 — 训练时同时预测多个下一个 token，提升训练效率
✅ DeepSeek-R1 纯 RL — 无 SFT 冷启动，纯强化学习激发推理，改变业界认知
✅ 极低成本 — V3 训练成本 ~$6M，颠覆"大模型要烧钱"的认知
```

### Kimi K2（Moonshot AI）
```
核心亮点：
✅ 1T 参数 MoE — 目前开源模型中参数量最大
✅ Muon 优化器 — 首次大规模使用 Muon（而非 AdamW），训练更稳定
✅ 超长上下文 — K2.5 支持 256K，原生长文本处理
✅ Agent Swarm — K2.5 支持 100 个并行子 Agent 协同
✅ 工具调用优化 — 专门为 Agentic 任务优化，工具调用成功率高
```

### GLM / ChatGLM（智谱 AI）
```
核心亮点：
✅ 国内最早开源中文对话模型（ChatGLM-6B，2023年）
✅ 轻量本地部署 — GLM-4-9B 仅需 5GB 显存，消费级可运行
✅ Agent 专项 — GLM-4.5 专门为 Agent 工具调用优化
✅ 与清华合作 — 学术界与工业界结合，中文理解深厚
```

### 豆包（字节跳动）
```
核心亮点：
✅ 字节生态整合 — 与抖音、飞书、剪映等深度整合
✅ 多模态全栈 — 文本+图像+视频+音频+图像生成（即梦 AI）
✅ 超低部署价格 — 豆包 API 价格是国内最低之一
✅ MarsCode — 代码专用模型，集成 VSCode 插件，对标 GitHub Copilot
❌ 不开源 — 无法本地部署或微调
```
