## 一、普通 Maven 项目对接

### 📋 对接流程

![image-20250623223418876](http://img.minalz.cn/typora/image-20250623223418876.png)

### 步骤 1：引入依赖

在 `pom.xml` 中添加 LangChain4j OpenAI 依赖：

```xml
<!-- LangChain4j OpenAI 集成 -->
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-open-ai</artifactId>
    <version>1.1.0</version>
</dependency>
```

🔗 [Maven 仓库地址](https://mvnrepository.com/artifact/dev.langchain4j/langchain4j-open-ai)

### 步骤 2：构建 OpenAiChatModel 对象

配置以下三个关键参数：

| 参数 | 说明 | 示例值 |
|:---|:---|:---|
| `baseUrl` | 阿里云百炼 API 地址 | `https://dashscope.aliyuncs.com/compatible-mode/v1` |
| `apiKey` | 你的 API 密钥 | 从百炼平台获取 |
| `modelName` | 模型名称 | `qwen-plus`、`qwen-turbo` 等 |

### 步骤 3：调用 chat 方法完成对话

使用 `model.chat()` 方法发送消息并获取回复。

### 步骤 4：引入日志依赖（可选）

为了查看请求和响应日志，需要引入 logback：

```xml
<!-- Logback 日志框架 -->
<dependency>
    <groupId>ch.qos.logback</groupId>
    <artifactId>logback-classic</artifactId>
    <version>1.5.18</version>
    <scope>compile</scope>
</dependency>
```

🔗 [Maven 仓库地址](https://mvnrepository.com/artifact/ch.qos.logback/logback-classic)

> 💡 **提示**：在构建 `OpenAiChatModel` 时设置 `logRequests(true)` 和 `logResponses(true)` 可以打印详细的请求和响应日志。

### 📝 完整 Demo 案例

```java
import dev.langchain4j.model.openai.OpenAiChatModel;

public class BaiLianDemo {
    public static void main(String[] args) {
        // 构建 OpenAiChatModel 对象
        OpenAiChatModel model = OpenAiChatModel.builder()
                .baseUrl("https://dashscope.aliyuncs.com/compatible-mode/v1")
                .apiKey(System.getenv("API-KEY"))  // 从环境变量读取 API Key
                .modelName("qwen-plus")
                .logRequests(true)      // 打印请求日志
                .logResponses(true)     // 打印响应日志
                .build();
    
        // 调用对话
        String result = model.chat("waylon帅不帅?");
        System.out.println(result);
    }
}
```

### 🔑 获取 API Key

1. 登录 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 进入 "API-KEY 管理" 页面
3. 创建或复制你的 API Key
4. 设置环境变量：
   ```bash
   # Windows (PowerShell)
   $env:API-KEY="your-api-key-here"
   
   # Linux/macOS
   export API-KEY="your-api-key-here"
   ```

## 二、SpringBoot 项目对接

### 📋 对接流程

![image-20250623223254356](http://img.minalz.cn/typora/image-20250623223254356.png)

### 步骤 1：创建 SpringBoot 项目

使用 [Spring Initializr](https://start.spring.io/) 创建项目，或使用 IDE 创建。

### 步骤 2：引入起步依赖

在 `pom.xml` 中添加 LangChain4j SpringBoot 集成依赖：

```xml
<!-- LangChain4j SpringBoot 起步依赖 -->
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-spring-boot-starter</artifactId>
    <version>1.1.0</version>
</dependency>

<!-- LangChain4j OpenAI 集成 -->
<dependency>
    <groupId>dev.langchain4j</groupId>
    <artifactId>langchain4j-open-ai-spring-boot-starter</artifactId>
    <version>1.1.0</version>
</dependency>
```

### 步骤 3：配置 application.yml

在 `application.yml` 中配置大模型参数：

```yaml
langchain4j:
  open-ai:
    chat-model:
      base-url: https://dashscope.aliyuncs.com/compatible-mode/v1
      api-key: ${BAI_LIAN_API_KEY}  # 从环境变量读取
      model-name: qwen-plus
      log-requests: true
      log-responses: true
      temperature: 0.7
      max-tokens: 2048
```

> 💡 **安全建议**：不要将 API Key 直接写在配置文件中，建议使用环境变量或配置中心。

### 步骤 4：开发接口，调用大模型

**创建 Controller：**

```java
import dev.langchain4j.model.openai.OpenAiChatModel;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api/ai")
public class ChatController {

    private final OpenAiChatModel chatModel;

    // 通过构造函数注入
    public ChatController(OpenAiChatModel chatModel) {
        this.chatModel = chatModel;
    }

    /**
     * 简单对话接口
     */
    @GetMapping("/chat")
    public String chat(@RequestParam String message) {
        return chatModel.chat(message);
    }

    /**
     * 带系统提示词的对话接口
     */
    @PostMapping("/chat-with-system")
    public String chatWithSystem(
            @RequestParam(defaultValue = "你是一个 helpful 的 AI 助手") String systemPrompt,
            @RequestParam String userMessage) {
        
        // 使用 UserMessage 和 SystemMessage
        String fullPrompt = systemPrompt + "\n\n用户问题：" + userMessage;
        return chatModel.chat(fullPrompt);
    }
}
```

**测试接口：**

```bash
# GET 请求
curl "http://localhost:8080/api/ai/chat?message=你好，请介绍一下自己"

# POST 请求
curl -X POST "http://localhost:8080/api/ai/chat-with-system" \
  -d "systemPrompt=你是一个技术专家" \
  -d "userMessage=什么是 LangChain4j？"
```

## 🎯 高级用法

### 1. 使用 AiServices（声明式方式）

**定义接口：**

```java
import dev.langchain4j.service.SystemMessage;
import dev.langchain4j.service.UserMessage;
import dev.langchain4j.service.AiService;

@AiService
public interface ChatAssistant {

    @SystemMessage("你是一个专业的技术顾问")
    @UserMessage("{{question}}")
    String answer(String question);
}
```

**使用服务：**

```java
@Service
public class AiServiceExample {
    
    private final ChatAssistant chatAssistant;

    public AiServiceExample(ChatAssistant chatAssistant) {
        this.chatAssistant = chatAssistant;
    }

    public String getTechAdvice(String topic) {
        return chatAssistant.answer("关于 " + topic + " 有什么建议？");
    }
}
```

### 2. 流式调用

```java
import dev.langchain4j.data.message.AiMessage;
import dev.langchain4j.model.StreamingResponseHandler;
import dev.langchain4j.model.openai.OpenAiStreamingChatModel;

// 创建流式模型
OpenAiStreamingChatModel streamingModel = OpenAiStreamingChatModel.builder()
        .baseUrl("https://dashscope.aliyuncs.com/compatible-mode/v1")
        .apiKey(System.getenv("BAI_LIAN_API_KEY"))
        .modelName("qwen-plus")
        .build();

// 流式调用
streamingModel.generate(
    "请写一篇关于 AI 的文章",
    new StreamingResponseHandler<AiMessage>() {
        @Override
        public void onNext(String token) {
            // 实时输出
            System.out.print(token);
        }

        @Override
        public void onComplete(Response<AiMessage> response) {
            System.out.println("\n完成！");
        }

        @Override
        public void onError(Throwable error) {
            error.printStackTrace();
        }
    }
);
```

### 3. 多模型切换

**配置多个模型：**

```java
@Configuration
public class ModelConfig {

    @Bean
    @Qualifier("qwenPlus")
    public OpenAiChatModel qwenPlus() {
        return OpenAiChatModel.builder()
                .baseUrl("https://dashscope.aliyuncs.com/compatible-mode/v1")
                .apiKey(System.getenv("BAI_LIAN_API_KEY"))
                .modelName("qwen-plus")
                .build();
    }

    @Bean
    @Qualifier("qwenTurbo")
    public OpenAiChatModel qwenTurbo() {
        return OpenAiChatModel.builder()
                .baseUrl("https://dashscope.aliyuncs.com/compatible-mode/v1")
                .apiKey(System.getenv("BAI_LIAN_API_KEY"))
                .modelName("qwen-turbo")
                .build();
    }
}
```

## ❓ 常见问题

### Q1: API Key 如何获取？

**解决方案：**
1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 完成实名认证
3. 进入 "API-KEY 管理" 页面
4. 创建新的 API Key

### Q2: 调用报错 401 Unauthorized？

**原因：** API Key 无效或未配置

**解决方案：**
- 检查 API Key 是否正确复制
- 确认 API Key 未过期
- 检查环境变量是否正确设置

### Q3: 调用超时怎么办？

**解决方案：**
```java
OpenAiChatModel model = OpenAiChatModel.builder()
        .baseUrl("https://dashscope.aliyuncs.com/compatible-mode/v1")
        .apiKey(System.getenv("BAI_LIAN_API_KEY"))
        .modelName("qwen-plus")
        .timeout(Duration.ofSeconds(60))  // 设置超时时间
        .build();
```

### Q4: 如何查看详细的请求和响应？

**解决方案：**
- 引入 logback 依赖
- 设置 `logRequests(true)` 和 `logResponses(true)`
- 查看控制台输出

### Q5: 支持哪些模型？

访问 [百炼平台文档](https://help.aliyun.com/zh/model-studio/getting-started/models) 查看最新支持的模型列表。

## 💡 最佳实践

### 1. 安全管理

- ✅ 使用环境变量存储 API Key
- ✅ 不要在代码中硬编码密钥
- ✅ 定期更换 API Key
- ✅ 使用配置中心管理密钥

### 2. 性能优化

- ✅ 复用 `OpenAiChatModel` 实例
- ✅ 合理设置 `maxTokens` 和 `temperature`
- ✅ 使用连接池（如需要）
- ✅ 添加缓存机制

### 3. 错误处理

```java
try {
    String result = model.chat(message);
    // 处理结果
} catch (Exception e) {
    // 记录日志

    // 返回友好提示

```
