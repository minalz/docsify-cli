# 🔑 实现两台服务器之间的免密登录

> 💡 Linux 服务器 SSH 免密登录配置 | SSH 密钥对 | 安全访问

---

## 📝 一、Master 服务器生成 SSH 密钥对

> 💡 **说明**：这个和生成 Git 的 SSH 秘钥是一模一样的，在 `~/.ssh/` 中。

### 1️⃣ 生成密钥对

```bash
ssh-keygen -t rsa
```

### 2️⃣ 查看公钥

```bash
cat id_rsa.pub
```

> 💡 **提示**：将公钥复制到另一台服务器中（Worker）。

---

## 🔄 二、切换到 Worker 服务器配置

### 1️⃣ 执行拷贝命令

在 Worker 服务器中执行下面的命令：

```bash
ssh-copy-id root@serverB_ip
```

### 2️⃣ 输入密码

输入 Master 服务器的密码，就可以实现免密登录了。

---

## ✅ 三、测试是否成功

输入命令，如果提示登录成功即可（未实现之前是需要输入 Master 密码才能登录成功的）：

```bash
ssh root@192.168.3.104
```
