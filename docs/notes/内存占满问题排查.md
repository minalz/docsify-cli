
---

## 🔧 二、清空内存缓存

发现 Swap 占用很高，清空一下内存：

```bash
sync
echo 3 > /proc/sys/vm/drop_caches

# 解释
# 0 - 不释放缓存
echo 0
# 1 - 释放页缓存
echo 1
# 2 - 释放 dentries 和 inodes 缓存
echo 2
# 3 - 释放 1 和 2 中说道的所有缓存
echo 3
```
